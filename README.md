# Visualization types

## Before each

2. Actions -> atoms/bonds -> show
3. Actions -> atoms/bonds -> stick

## Backbone
1. Actions -> ribbon -> hide
2. Actions -> atoms/bonds -> stick
3. Actions -> atoms/bonds -> backbone only -> chain trace
![Backbone](/backbone.png)

## Spacefill

2. Actions -> atoms/bonds -> show
3. Actions -> atoms/bonds -> sphere
